# jwt-auth
JWT auth flow example for server and client, for educational purpose


#### Server (Express)
##### Add .env

```
PORT=4000
ACCESS_TOKEN_SECRET=youraccesstokensecret
REFRESH_TOKEN_SECRET=yourrefreshtokensecret
```
##### then 
```
npm install && npm run dev
```

#### Client (React)

 
```
npm install && npm start
```
